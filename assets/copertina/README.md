# Copertina

Inserisci qui il file `copertina.jpg` (o `.png`).

Requisiti consigliati:
- Dimensioni: 1600 × 2400 px (ratio 2:3)
- Formato: JPEG o PNG
- Risoluzione: 150–300 dpi

`build.sh` lo include automaticamente se il file si chiama `copertina.jpg`.
Per usare un file diverso, modifica la variabile `COVER` in `build.sh`.
